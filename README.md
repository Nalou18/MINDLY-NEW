# MINDLY — Watch. Protect. Care.

Prototype React + Vite + Tailwind + Recharts.

## Développement local
```bash
npm install
npm run dev
```

## Déploiement Vercel
- Importez ce dossier dans un repo GitHub ou zippez-le et uploadez sur Vercel.
- Vercel détecte Vite automatiquement.
- Build command: `npm run build`
- Output directory: `dist`

## Notes
- Le logo est dans `src/assets/mindly-logo.png` et utilisé automatiquement.
- L'indice MINDLY n'agrège **que** les réponses organisationnelles (Gollac).
